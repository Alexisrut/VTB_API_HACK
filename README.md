# FinFlow - Financial Management Platform

Мультибанковская платформа для управления финансами с аналитикой и прогнозированием денежных потоков.

## Описание

FinFlow - это веб-приложение для предпринимателей и малого бизнеса, которое позволяет:

- **Мультибанкинг** - объединять счета из разных банков в одном интерфейсе
- **Финансовая аналитика** - отслеживать доходы, расходы и ключевые метрики
- **Прогнозирование денежных потоков** - ML-предсказания кассовых разрывов
- **Управление дебиторской задолженностью** - контроль счетов к получению
- **Контрагенты** - ведение базы клиентов и поставщиков

## Технологии

### Backend
- **FastAPI** - Python веб-фреймворк
- **SQLAlchemy** - ORM для работы с базой данных
- **PostgreSQL** - основная база данных
- **Redis** - кэширование и Celery задачи
- **JWT** - аутентификация
- **OAuth2** - интеграция с банками

### Frontend
- **React 18** - пользовательский интерфейс
- **TypeScript** - типизация
- **Vite** - сборка
- **SCSS Modules** - стилизация
- **Lucide React** - иконки

### Инфраструктура
- **Docker** - контейнеризация
- **Docker Compose** - оркестрация
- **Nginx** - прокси и статика

## Быстрый старт

### Требования
- Docker
- Docker Compose

### Запуск

```bash
# Клонировать репозиторий
git clone <repository-url>
cd finflow

# Запуск в режиме разработки
./docker.sh start

# Или запуск продакшн версии
./docker.sh start prod
```
